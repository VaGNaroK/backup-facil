Como gerar o seu .exe.

Passo 1: Levar o código para o Windows
Vá para o seu ambiente Windows e abra o terminal (CMD ou PowerShell). Como o projeto já está no seu GitHub com a nova estrutura, basta clonar:

git clone https://github.com/VaGNaroK/backup-facil.git
cd backup-facil

Passo 2: Preparar a "Bolha" no Windows
Dentro da pasta do projeto no Windows, crie o ambiente virtual e instale as bibliotecas. Como agora temos um arquivo oficial de dependências, você não precisa mais digitar o nome de cada biblioteca:

python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt

Passo 3: O Script Mágico do Windows (gerar_exe.bat)
Crie (ou renomeie o seu txt) para um arquivo chamado gerar_exe.bat dentro da pasta scripts\ e cole este código. Ele já está configurado com os caminhos corretos (src\main.py, assets\icon.ico) e com o separador do Windows (;):

@echo off
echo ==========================================
echo   Gerando o Executavel do Backup Facil Pro
echo ==========================================

echo [1] Limpando arquivos e caches antigos...
if exist "build" rmdir /s /q "build"
if exist "dist" rmdir /s /q "dist"
del /q *.spec 2>nul

echo [2] Iniciando o PyInstaller (PySide6)...
python -m PyInstaller --noconsole --onefile --name "Backup_Facil_Pro" --icon="assets\icon.ico" --add-data "assets;assets" src\main.py

echo [3] Limpeza final profunda...
if exist "build" rmdir /s /q "build"
del /q *.spec 2>nul
for /d /r . %%d in (__pycache__) do @if exist "%%d" rd /s /q "%%d"

echo.
echo ==========================================
echo SUCESSO! O arquivo Backup_Facil_Pro.exe esta na pasta 'dist'!
echo ==========================================
pause

Passo 4: Compilar!
Ainda com o seu (venv) ativado no terminal do Windows, certifique-se de estar na raiz do projeto e rode o script chamando a pasta scripts:

scripts\gerar_exe.bat

agora deixe o Windows compilar toda a interface do PySide6 e a criptografia AES-256. Quando terminar, você terá uma pasta chamada dist\ e lá dentro estará o seu Backup_Facil_Pro.exe.
